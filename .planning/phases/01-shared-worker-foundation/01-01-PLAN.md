  <action>
    1. Create `worker-template/tests/test_audio_utils.py` with RED tests matching all behaviors above. Use `sample_audio_bytes` fixture from conftest. For URL tests, mock `requests.get` to return a response object with `iter_content()` yielding sample bytes and `headers` with appropriate content-type. For SSRF tests, mock `socket.getaddrinfo` to return the target IP.

    2. Create `worker-template/audio_utils.py` implementing:

       **_validate_url(url: str) -> None:**
       - Parse URL with urllib.parse.urlparse
       - Check scheme in {"http", "https"} -- raise ValueError on others
       - Check hostname is not None
       - Resolve hostname via socket.getaddrinfo, extract IP
       - Check ip_address is not is_private, is_loopback, or is_link_local
       - Raise ValueError with descriptive message on any failure

       **_detect_extension(url: str, content_type: str) -> str:**
       - Try to extract extension from URL path (using os.path.splitext on urlparse(url).path)
       - If valid extension found in ALLOWED_EXTENSIONS set (.wav, .mp3, .m4a, .ogg, .flac, .webm, .mp4), return it
       - Fall back to content-type mapping: audio/wav->.wav, audio/mpeg->.mp3, audio/mp4->.m4a, audio/ogg->.ogg, audio/flac->.flac
       - Default to ".wav" if nothing detected

       **resolve_audio_input(job_input: dict) -> str:**
       - If "audio_url" in job_input: validate URL, download with requests.get(timeout=120, stream=True), detect extension, write to NamedTemporaryFile(suffix=ext, delete=False), return path
       - If "audio_base64" in job_input: base64.b64decode, write to NamedTemporaryFile(suffix=".wav", delete=False), return path
       - Else: raise ValueError("Provide 'audio_url' or 'audio_base64'")

       **cleanup_audio(path: str) -> None:**
       - try os.unlink(path), except OSError: pass

       **encode_audio_output(audio_array, sample_rate: int, format: str = "wav") -> dict:**
       - Write audio_array to BytesIO using soundfile.write(buf, audio_array, sample_rate, format=format)
       - base64 encode the bytes
       - Return dict with audio_base64, format, sample_rate, duration_seconds (len(audio_array)/sample_rate)
